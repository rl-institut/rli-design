Colors

Primary color is $blue. It is mostly used for text, but also some background elements images.

Color for anchors is $link-color.

Try to use very light background colors such as $white or $gray-100. You can use box-shadows such as $box-shadow-sm on a component if the component and background both have $white as a background-color.
